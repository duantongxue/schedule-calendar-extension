# 排班日历助手 Chrome 扩展

一个简洁实用的排班日历查看工具，支持白班、小夜、大夜的循环排班显示。

## 功能特点

- 📅 查看任意月份的排班情况
- 🔄 支持自定义基准日期和对应班次
- 💾 自动保存设置到本地存储
- 🎨 响应式设计，适配不同屏幕尺寸
- 🌙 清晰的班次颜色区分（白班/小夜/大夜）
- ⚡ 快速切换月份查看

## 安装方法

### 开发模式安装：

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的"开发者模式"开关
3. 点击"加载已解压的扩展程序"按钮
4. 选择本项目所在的文件夹（shift-calendar-extension）
5. 扩展将出现在浏览器工具栏中

### 使用方法：

1. 点击浏览器工具栏中的扩展图标打开排班日历
2. 在顶部设置区设置**基准日期**和对应的**班次**
   - 基准日期：选择一个你知道确切班次的日期
   - 班次：选择该日期对应的班次（白班/小夜/大夜）
3. 点击"保存"按钮，系统会自动跳转到基准日期所在月份
4. 使用日历上方的左右箭头（❮ ❯）切换查看不同月份
5. 今天的日期会以蓝色边框高亮显示

## 排班计算逻辑

以设定的基准日期为起点，按照 3 天一个循环（白班 → 小夜 → 大夜）自动推算任意日期的班次：

- **白班**：黄色标签
- **小夜**：青色标签  
- **大夜**：红色标签

例如：如果设置 2026-02-25 为白班，那么：
- 2026-02-26：小夜
- 2026-02-27：大夜
- 2026-02-28：白班
- 以此类推...

## 自定义配置

### 修改默认基准日期

编辑 `popup.js` 文件第 4 行：

```javascript
let config = { baseDate: new Date("2026-02-25"), baseShiftIndex: 0 };
```

- 修改日期为你需要的基准日期
- `baseShiftIndex`: 0=白班，1=小夜，2=大夜

### 调整班次颜色

编辑 `index.html` 文件中的 CSS 部分（第 159-172 行）：

```css
.shift-白班 {
    background: #fff3cd;
    color: #856404;
}
/* 修改这里的颜色值 */
```

### 修改班次名称

编辑 `popup.js` 文件第 2 行：

```javascript
const shiftNames = ["白班", "小夜", "大夜"];
// 可以改为你实际的班次名称，如 ["早班", "中班", "晚班"]
```

## 注意事项

1. **图标文件**：当前 `icons/` 文件夹为空，需要自行准备三个尺寸的 PNG 图标（16x16, 48x48, 128x128），或者暂时忽略（扩展仍可正常运行）

2. **数据存储**：设置信息保存在浏览器的 LocalStorage 中，清除浏览器数据时可能会丢失

3. **兼容性**：仅支持 Chrome 88+ 及其他支持 Manifest V3 的 Chromium 内核浏览器（如 Edge、Brave 等）
